# Changelog

All notable changes to the Instagres VS Code extension will be documented in this file.

## [0.1.0] - 2025-12-19

Initial pre-release.

### Features

- Create instant Neon PostgreSQL databases via Instagres API
- Dual connection string support (pooled and direct)
- Quick action buttons for copying connection strings and opening claim URL
- Database history with expiration tracking
- Zero configuration required
- 72-hour database lifespan (claimable for permanent use)
